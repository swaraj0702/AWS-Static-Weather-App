# AWS Static Weather App

## Overview
A responsive static Weather Application deployed on AWS using Amazon S3 and CloudFront,
providing secure HTTPS access and real-time weather data.

## Technologies Used
- Amazon S3 (Static Website Hosting)
- Amazon CloudFront (HTTPS & CDN)
- HTML, CSS, JavaScript
- OpenWeather API

## Architecture
Users → CloudFront → S3 Bucket

## Features
- Responsive UI
- Secure HTTPS connection
- Real-time weather updates
- Cloud-based deployment
